Iris Flower Clustering using K-Means

This project applies **K-Means Clustering (Unsupervised Machine Learning)** to group Iris flowers into 3 clusters based on four features:

- Sepal Length  
- Sepal Width  
- Petal Length  
- Petal Width  

The goal is to see how well K-Means can naturally form clusters matching the three species.

Project Overview
This project includes:
- Loading the Iris dataset  
- Preprocessing and scaling  
- Applying K-Means algorithm  
- Visualizing clusters  
- Comparing clusters with real species labels
